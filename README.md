# Vim / Neovim

## Installation

For now just copy it over (vim user) and create a symbolic link (neovim)

```
cp config/.vimrc ~/
ln -s ~/.vimrc ~/.config/nvim/init.vim
```

## Dependencies

I use this [vim-plug](https://github.com/junegunn/vim-plug) as my package manager.
